Scavenger Cloud - 垃圾佬云
--------------------------

Scavenger Cloud - 垃圾佬云. 这个冬天太寒冷, 为了提升家里的室温, 广大垃圾佬决定贡献自己的CPU为大家提供算力, 顺便利用CPU取暖.  
垃圾佬云最终的目的是建立一个群体共有的利用虚拟货币计费的私有云.
